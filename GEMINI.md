# GEMINI.md

## Project Overview
**Project Name:** DevCmds
**Project Type:** React (TypeScript) Web Application

A searchable, copy-to-clipboard command-line dashboard designed for full-stack and AI-oriented developers.

## Building and Running
*   **Build Command:** `npm run build`
*   **Run Command:** `npm run dev`
*   **Test Command:** `npm run lint`

## Development Conventions
*   **Language/Framework:** React 18, TypeScript, Vite.
*   **Styling:** Vanilla CSS with custom properties for theme management.
*   **Icons:** Lucide React.
*   **Deployment:** Configured for GitHub Pages via `vite.config.ts` (`base: './'`).

## Key Files
*   `src/App.tsx`: Main application logic, command data, and filtering system.
*   `src/App.css`: Core application styling (glassmorphism theme).
*   `README.md`: Setup and deployment instructions.
